# Переводчик модов Minecraft на русский язык

Программа для автоматического перевода модов Minecraft на русский язык с использованием машинного перевода.

## Возможности

- ✅ Автоматический поиск файлов переводов в модах
- ✅ Поддержка стандартной структуры модов (Forge/Fabric)
- ✅ Сохранение форматирования и структуры JSON
- ✅ Прогресс-бар для отслеживания процесса
- ✅ Статистика перевода
- ✅ Обработка ошибок и пропуск технических строк

## Установка

1. Убедитесь, что у вас установлен Python 3.7 или выше
2. Установите зависимости:

**Windows (простой способ):**
Просто запустите файл `install.bat` двойным кликом!

**Или вручную через командную строку:**

```bash
# Для Windows используйте команду py:
py -m pip install -r requirements.txt

# Или если у вас настроен python:
python -m pip install -r requirements.txt
```

## Использование

### Базовое использование

Перевести мод, указав путь к папке мода:

**Windows:**
```bash
py translator.py "путь\к\папке\мода"
```

**Или если настроен python:**
```bash
python translator.py "путь/к/папке/мода"
```

### Сохранение в другую папку

```bash
py translator.py "путь\к\папке\мода" -o "путь\для\сохранения"
```

### Изменение языков перевода

```bash
py translator.py "путь\к\папке\мода" --source-lang en --target-lang ru
```

## Примеры

### Пример 1: Перевод мода из папки

```bash
py translator.py "C:\Users\konos\AppData\Roaming\.minecraft\mods\MyMod"
```

### Пример 2: Перевод с сохранением в отдельную папку

```bash
py translator.py "C:\Users\konos\mods\MyMod" -o "C:\Users\konos\mods\MyMod_RU"
```

## Структура файлов переводов

Программа автоматически находит файлы переводов в следующих местах:
- `assets/*/lang/en_us.json` (стандартная структура Forge/Fabric)
- `lang/en_us.json` (старые моды)
- `lang/en.json` (альтернативный формат)

Переведенные файлы сохраняются как `ru_ru.json` в тех же папках.

## Как это работает

1. Программа сканирует указанную папку мода
2. Находит все файлы переводов (обычно `en_us.json`)
3. Читает JSON файлы и извлекает тексты для перевода
4. Использует Google Translate API для перевода текстов
5. Сохраняет переведенные файлы как `ru_ru.json`

## Ограничения

- Программа не переводит технические строки (цветовые коды, форматирование)
- Требуется подключение к интернету для работы с API перевода
- Google Translate имеет лимиты на количество запросов (бесплатно ~100,000 символов в день)

## Работа с .jar файлами

### Автоматический перевод .jar файла

Используйте утилиту `jar_handler.py` для автоматического перевода .jar файлов:

```bash
py jar_handler.py translate "путь\к\моду.jar"
```

### Ручная работа с .jar файлами

Если нужно вручную распаковать/упаковать .jar:

```bash
# Распаковать .jar файл
py jar_handler.py extract "мод.jar" -o "папка_для_распаковки"

# Упаковать папку обратно в .jar
py jar_handler.py pack "папка_мода" -o "мод_переведенный.jar"
```

### Старый способ (вручную)

Если нужно переводить .jar файлы вручную:
1. Переименуйте `.jar` в `.zip`
2. Распакуйте архив
3. Используйте программу на распакованной папке
4. Запакуйте обратно в `.jar`

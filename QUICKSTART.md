# Быстрый старт

## 1. Установка зависимостей

**Простой способ:** Запустите файл `install.bat` двойным кликом!

**Или вручную:**
```bash
py -m pip install -r requirements.txt
```

## 2. Использование

### Перевод папки с модом:
```bash
py translator.py "C:\путь\к\папке\мода"
```

### Перевод .jar файла мода:
```bash
py translator.py "C:\путь\к\моду.jar"
```

### Перевод с сохранением в другую папку:
```bash
py translator.py "C:\путь\к\моду" -o "C:\путь\для\сохранения"
```

## 3. Windows - двойной клик

Используйте файл `translate_mod.bat` - просто перетащите на него папку или .jar файл мода!

## Что происходит?

1. Программа находит все файлы `en_us.json` в моде
2. Переводит тексты на русский язык
3. Сохраняет как `ru_ru.json` в тех же папках

## Где найти переведенные файлы?

Переведенные файлы находятся в тех же папках, что и исходные, но с именем `ru_ru.json`:
- Было: `assets/mymod/lang/en_us.json`
- Стало: `assets/mymod/lang/ru_ru.json`

## Проблемы?

- **Нет интернета?** Программа требует подключение к интернету для работы с Google Translate
- **Ошибка импорта?** Убедитесь, что установлены все зависимости: запустите `install.bat` или `py -m pip install -r requirements.txt`
- **Не находит файлы?** Проверьте, что в моде есть папка `assets` или `lang` с файлами переводов
- **Команда python не работает?** Используйте `py` вместо `python` в Windows
